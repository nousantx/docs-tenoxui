# Installation

Add tenoxui to your project. You can install it using npm or simply attach the file hosted on CDN.

## NPM

NPM installation

<Code
  lang="bash"
  className="my-1rem" codeOnly>{`npm i tenoxui@latest --save-dev`}</Code>

## CDN

<Code
  lang="html"
  className="my-1rem" codeOnly>{`<script src="https://cdn.jsdelivr.net/npm/tenoxui"></script>`}</Code>

## React Environment

### Install

Add TenoxUI into your react project.

<Code
  lang="bash"
  className="my-1rem" codeOnly>{`npm i tenoxui --save-dev`}</Code>
